## Closed Deals Workflow

This workflow will automatically update the progress of your Key Result based on the number of closed deals in Hubspot within the time the Objective is set for. It will count the number of closed deals and update the Key Result progress accordingly.

## Win Rate Workflow

This workflow will automatically update the progress of your Key Result based on the win rate (deals closed / total deals) of deals in Hubspot within the time the Objective is set for. It will calculate the win rate percentage and update the Key Result progress accordingly.
